# You probably meant `@ar-agents` (plural)

This package is a **defensive squat**. The canonical scope on npm is `@ar-agents` (plural, with a hyphen). Common typos go to this stub so a stranger can't ship a back-doored package under a confusingly similar name.

## Install the real thing

```sh
pnpm add @ar-agents/<package>
```

Examples:

```sh
pnpm add @ar-agents/identity
pnpm add @ar-agents/mercadopago
pnpm add @ar-agents/iibb
pnpm add @ar-agents/wscdc
pnpm add @ar-agents/tienda-nube
# … and 15+ more
```

## Why

The `@ar-agents` scope ships open-source agent tooling for Argentine regulatory + commerce integrations. We register typo-adjacent scopes ourselves to keep the npm namespace clean. Read more at <https://ar-agents.ar>.
