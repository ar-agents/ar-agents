# apps/studio

Conversational builder: chat an idea into an operating automated society (sociedad automatizada), on top of `@ar-agents/*`.
Agent backend lives at `/api/agent`. See `docs/CONTRACT.md` for the exact API contract (accounts, metering, agent, society lifecycle).
See root `ROADMAP.md` items M0-1 through M0-6 for the build order.
This version of Next.js differs from training data: read `node_modules/next/dist/docs/` before writing code.
